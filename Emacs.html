Navigating within the file

Packages:

  • Built-in
    • Outline mode For indentation-based languages (Python).
    • Hideshow mode For languages where blocks are delimited by braces (Emacs Lisp, C).
    • Narrow functions
    • Comment functions
    • Xref
    • Org
  • External
    • Hl-todo
    • Consult-todo
    • Evil

Overview of the bindings:

  • M- narrows
  • C- folds
  • º selects element
  • + shows everything
  • - hides everything
  • SPC ' searches for markers in the buffer
  • SPC t deals with TODO
  • SPC s deals with searches
